# **App Name**: SegurUlo

## Core Features:

- Landing Page: Visually appealing landing page showcasing the smart helmet features and benefits, including a hero section, helmet mockup, feature highlights, testimonials, and footer.
- User Authentication: Secure user authentication system with email/password sign-up and sign-in, including social login option. Stores credentials safely.
- Helmet Connection Status: Real-time display of helmet connection status (Connected/Disconnected).
- Emergency Contact Management: System to add, remove, and manage emergency contacts.
- Settings Customization: Allows users to customize helmet settings and preferences. The LLM will use reasoning as a tool to choose personalized settings.
- Emergency Alert System: Allows users to trigger emergency alerts and notify their added emergency contacts.

## Style Guidelines:

- Primary color: Dark, desaturated blue (#3B5998) to evoke a sense of trust and security, aligning with the protective nature of the helmet.
- Background color: Very dark blue (#1A1A2E) to create a professional and secure environment.
- Accent color: Light blue (#FF7F50) to highlight key interactive elements and add a touch of modernity.
- Font: 'Poppins' sans-serif font known for its geometric shapes that provide clean and readable text on both desktop and mobile. Note: currently only Google Fonts are supported.
- Mobile-first, responsive design, ensuring a seamless experience across different screen sizes. Rounded corners (2xl) and soft shadows to add depth and a modern feel to the UI.
- Subtle Framer Motion animations for button presses and transitions to provide a smooth and engaging user experience.
